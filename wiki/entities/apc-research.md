# apc-research

Windows **APC internals** research code (repnz): sample programs plus a shared library of common helpers for studying Asynchronous Procedure Call behavior. Aimed at game-security researchers and reverse engineers in the cheat / Windows kernel explorer lane—useful background when reading kernel APC injectors such as [[injdrv]] / [[kinject]] or AC thread/APC inspection.
