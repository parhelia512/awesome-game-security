# ida-plugins

IDA Pro plugin providing **register cross-references** for static RE. Contrasts with Oregami-style value-use analysis that filters to references sharing the same value; this project surfaces register xrefs more directly. Useful for game-security researchers in the cheat / IDA Plugins lane. (source: wiki/sources/descriptions/repnz__ida-plugins.md)

Not a curated plugin pack—single-purpose register-xref tooling. Complements broader xref helpers such as [[xrefsext]] and collection packs like [[idaplugins]].
