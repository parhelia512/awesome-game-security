# ilhook-rs

Rust x86 HOOK library: intercepts function calls and routes them to user-defined handlers. Aimed at game-security researchers and reverse engineers studying offensive user-mode hook/trampoline techniques in the cheat / hook lane. Complements C++ hook libs such as [[polyhook]], [[polyhook-2-0]], and [[detoursnt]]. (source: wiki/sources/descriptions/regomne__ilhook-rs.md)
