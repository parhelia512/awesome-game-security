VoltClient — внешний C++ чит-клиент для AssaultCube, который подключается к процессу игры и накладывает ESP и помощь в прицеливании. Он использует Windows API работы с памятью процессов (ReadProcessMemory и WriteProcessMemory) для ac_client.exe с задокументированными смещениями для версии 1.3.0.2, а также преобразование мир–экран для отрисовки. Интерфейс — прозрачный оверлей на DirectX 11 с Dear ImGui: ESP-боксы, полоски здоровья, трассеры, круг FOV и сглаженный aimbot на основе FOV, включаемый через внутриигровое меню. Предназначен для исследователей игровой безопасности, реверс-инженеров и учащихся, изучающих внешние оверлеи, чтение памяти и техники, связанные с античитом, на простом FPS.
