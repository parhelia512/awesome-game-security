VoltClient 是一款面向 AssaultCube 的外部 C++ 作弊客户端，可附加到游戏进程并叠加 ESP 与瞄准辅助。它通过 Windows 进程内存 API（ReadProcessMemory 与 WriteProcessMemory）操作 ac_client.exe，并使用针对 1.3.0.2 版本的已文档化偏移，以及用于渲染的世界坐标到屏幕坐标换算。界面为基于 DirectX 11 的透明叠加层，配合 Dear ImGui，提供 ESP 方框、血条、射线追踪、FOV 圆圈，以及可通过游戏内菜单开关的平滑 FOV 自瞄。面向游戏安全研究者、逆向工程师，以及希望在简单 FPS 上学习外部叠加、内存读取与反作弊相关技术的学习者。
