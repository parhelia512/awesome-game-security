VoltClient는 AssaultCube용 외부 C++ 치트 클라이언트로, 게임 프로세스에 연결하여 ESP와 조준 지원을 오버레이합니다. Windows 프로세스 메모리 API(ReadProcessMemory 및 WriteProcessMemory)로 ac_client.exe를 다루며, 버전 1.3.0.2용으로 문서화된 오프셋과 렌더링용 월드-투-스크린 변환을 사용합니다. UI는 DirectX 11 투명 오버레이와 Dear ImGui로 구성되며, ESP 박스, 체력 바, 트레이서, FOV 원, 인게임 메뉴로 전환하는 부드러운 FOV 기반 에임봇을 제공합니다. 단순한 FPS에서 외부 오버레이, 메모리 읽기, 안티치트 관련 기법을 연구하는 게임 보안 연구자, 리버서, 학습자를 대상으로 합니다.
