VoltClient 是一款面向 AssaultCube 的外部 C++ 作弊用戶端，可附加至遊戲行程並疊加 ESP 與瞄準輔助。它透過 Windows 行程記憶體 API（ReadProcessMemory 與 WriteProcessMemory）操作 ac_client.exe，並使用針對 1.3.0.2 版本的已文件化偏移，以及用於繪製的世界座標到螢幕座標轉換。介面為基於 DirectX 11 的透明疊加層，搭配 Dear ImGui，提供 ESP 方框、血條、軌跡線、FOV 圓圈，以及可透過遊戲內選單開關的平滑 FOV 自瞄。面向遊戲安全研究者、逆向工程師，以及希望在簡單 FPS 上學習外部疊加、記憶體讀取與反作弊相關技術的學習者。
