VoltClient は AssaultCube 向けの外部 C++ チートクライアントで、ゲームプロセスにアタッチして ESP とエイム支援をオーバーレイ表示します。Windows のプロセスメモリ API（ReadProcessMemory と WriteProcessMemory）を用いて ac_client.exe を操作し、バージョン 1.3.0.2 向けに文書化されたオフセットと、描画用のワールド・トゥ・スクリーン変換を利用します。UI は DirectX 11 の透明オーバーレイと Dear ImGui で構成され、ESP ボックス、体力バー、トレーサー、FOV 円、ゲーム内メニューで切り替え可能な平滑 FOV ベースのエイムボットを提供します。シンプルな FPS で外部オーバーレイ、メモリ読み取り、アンチチート関連手法を学ぶゲームセキュリティ研究者、リバースエンジニア、学習者を対象としています。
